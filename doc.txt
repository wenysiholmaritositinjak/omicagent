 1.1 项目背景与定位
    植物单细胞与空间转录组学是当前植物功能基因组学研究的前沿方向。随着 snRNA-seq、
 spatial RNA-seq 等技术的成熟，植物研究正在从组织水平解析走向单细胞精度的基因表达图谱构建。
 然而，数据获取、环境搭建与跨数据集整合的全流程仍高度依赖人工操作，科研人员 60% 以上的时间
 消耗在数据准备与环境维护上，严重制约了科学假设验证的效率。

    本项目旨在构建面向植物组学的 AI Scientist Agent——OmicAgent，依托 DCS Cloud 平台算
 力、OmicSeek 数据检索接口与训推平台大模型 API，实现从自然语言驱动的数据检索、环境自动搭
 建、元数据语义解析到跨物种整合分析的全流程自动化。项目以 AI+X 创新应用大赛为起点，目标在
 2 个月内完成从原型验证到院内推广的完整落地路径。

 3.2.2 工具调度器（Tool Dispatcher）
      工具调度器是 Agent 与外部资源交互的统一网关。它封装了三类工具的调用逻辑：（1）
 OmicSeek 数据检索接口——将自然语言查询转换为 OmicSeek API 参数，解析返回的 JSON 结果；
 （2）DCS Cloud 容器命令接口——在项目空间内执行 Shell 命令、管理文件、启动分析任务；（3）
 训推平台大模型接口——以统一的 HTTP 客户端封装不同模型的调用，支持自动重试、Token 计数与
 Context Cache。

 3.2.4 报告生成器（Report Generator）
    报告生成器将分析结果汇编为可交互的 HTML 报告，包含以下核心组件：（1）分析摘要——由
 大模型对分析结果生成自然语言总结；（2）关键图表——UMAP 降维图、差异表达热图、共表达模块
 网络图等，以可交互的 Plotly 图表嵌入；（3）数据表格——标记基因列表、细胞类型比例统计等，以
 可排序的 DataTable 呈现；（4）可复现代码——每步分析对应的脚本代码块，支持一键复制。报告
 模板使用 Jinja2 引擎渲染，可在 DCS Cloud 项目空间内直接预览。
